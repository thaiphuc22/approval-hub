# 🔍 ba-doc-review — Review tài liệu nghiệp vụ chuẩn IIBA

> Review BRD/FRD/SRS/User Story/Use Case → phát hiện gaps, ambiguity,
> INVEST violations, traceability issues, MoSCoW imbalance, compliance gaps.

## 📋 Skill này làm gì?

Đóng vai **Senior BA Reviewer** chuẩn IIBA CBAP, review độc lập tài liệu
nghiệp vụ với:

- **7 dimensions check**: Completeness, Clarity, Consistency, Testability,
  Traceability, Prioritization, Compliance
- **Ambiguity scan**: phát hiện 60+ từ mơ hồ ("nhanh", "dễ dùng", "phù hợp",
  "tối ưu"...) bằng dictionary chuẩn
- **INVEST validation** cho user story (6/6 tiêu chí)
- **MoSCoW balance check** (cảnh báo nếu Must >60%)
- **Compliance gap check** (Nghị định 13, GDPR, PCI-DSS, HIPAA)
- **Severity-based scoring** (Critical 20pt, Major 5pt, Minor 1pt)
- **Grade S/A/B/C/D/F** với roadmap fix cụ thể

**Output 2 formats:**
- **A: Markdown Report** — findings ranked + suggested fixes + summary score
- **B: Inline Annotated** — file gốc + HTML comments tại từng vị trí lỗi

## 📊 Severity Levels

| Severity | Icon | Khi nào | Score impact |
|---|---|---|---|
| **Critical** | 🔴 | Doc unusable / compliance violation / missing AC | -20pt |
| **Major** | 🟠 | Ambiguity / INVEST fail / orphan requirement | -5pt |
| **Minor** | 🟡 | Polish / style / naming | -1pt |
| **Info** | ℹ️ | Optional improvement | 0pt |

**Compliance-heavy weight** (payment/health/PII): Critical = ×2 (40pt trừ).
