## v1.0.0 (2026-05-09)

🎉 **Initial release**

### Features
- Review BRD/FRD/SRS/User Story/Use Case theo chuẩn IIBA BABOK v3
- 7-dimension scoring: Completeness, Clarity, Consistency, Testability,
  Traceability, Prioritization, Compliance
- Ambiguity dictionary 60+ từ Vietnamese + English
- INVEST validation cho user story (6 criteria)
- MoSCoW balance check (warning >60% Must, fail >80%)
- Compliance gap detection (Nghị định 13, GDPR, PCI-DSS)
- Severity-based scoring (Critical/Major/Minor/Info)
- Grade S/A/B/C/D/F với roadmap fix
- 2 output formats:
  - **A:** Markdown report (default) — ranked findings + suggested fixes
  - **B:** Inline annotated (HTML comments tại line lỗi)
- Compliance-heavy weight (×2) cho doc payment/health/PII

### Scripts
- `scripts/doc_tool.py` — 2 subcommands:
  - `parse` — Parse `.docx`/`.md`/`.txt` → structured JSON
  - `annotate` — Inline-annotate file gốc với findings
- Dependencies: `python-docx>=1.1.0` (chỉ cần khi parse `.docx`)
