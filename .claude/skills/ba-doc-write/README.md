# 📝 ba-doc-write — Sinh tài liệu nghiệp vụ chuẩn IIBA

> Sinh BRD, FRD, SRS, User Story (INVEST), Use Case theo chuẩn BABOK v3 — Agile/Scrum perspective.

## 📋 Skill này làm gì?

Là Senior BA chuẩn IIBA CBAP đóng vai trò sinh tài liệu nghiệp vụ chất lượng:

- **BRD** — Business Requirements Document (Why)
- **FRD** — Functional Requirements Document (What)
- **SRS** — Software Requirements Specification (BRD + FRD + NFR)
- **User Story** — chuẩn INVEST + Acceptance Criteria Given-When-Then
- **Use Case** — main flow + alternative + exception flows

Mọi requirement có **ID + MoSCoW Priority + Acceptance Criteria + Traceability**.
