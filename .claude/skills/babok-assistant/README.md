# 🎯 babok-assistant — Trợ lý BA + Orchestrator

> Senior BA coach + traffic router. Phân loại intent → điều hướng tới
> skill phù hợp (`ba-doc-write` / `ba-doc-review`) hoặc handle inline
> (BABOK knowledge / workflow coach).

## 📋 Skill này làm gì?

Đóng 3 vai trò chính:

### 1. 🚦 Router
- Nhận 1 câu input của BA → phân loại intent (8 categories) → route
  tới skill chuyên xử lý hoặc handle inline.
- Pass đầy đủ context khi route → user không phải kể lại.

### 2. 📚 Coach inline
Handle các case không cần worker skill:
- **Tư vấn BABOK knowledge** (BACCM, MoSCoW, INVEST, SWOT, technique)
- **Workaround skill chưa có** (`ba-elicit`, `ba-strategy`, `ba-solution-eval`)
- **Methodology decision** (Agile vs Waterfall vs Hybrid)

### 3. 🎯 Workflow Orchestrator
Khi user có task lớn (multi-phase project):
- Map task → BABOK 6 KAs → skill sequence
- Estimate timeline + identify ready vs roadmap skills
- Coach BA junior step-by-step
