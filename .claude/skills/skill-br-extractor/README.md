## 🔌 Input/Output

**Input:** policy text / SOP / regulation document

**Output:** JSON với:
- `business_rules[]` — normalized với condition + action + source
- `decision_table` — nếu ≥3 conditions
- `conflicts_detected[]` — flag for human review
- `coverage_gaps[]` — missing combinations
